In this folder, one needs to add the datasets corresponding to the applications at hand. 

The datasets that are used within this code come from the following sources:

  1) PDE optimization datasets:
        https://personalpages.manchester.ac.uk/staff/david.silvester/ifiss/default.htm
     Please note that for the PDE case, the problems are generated using function that can 
     be found in the above link.
        
  2) Linear regression datasets:
        https://www.csie.ntu.edu.tw/~cjlin/libsvm/
        
 The data are not included in this file.
